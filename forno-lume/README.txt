Pizzeria Forno Lume — editorial menu style demo with language switcher

How to open:
1. Unzip the folder.
2. Open index.html in your browser.

Files:
- index.html — page structure
- styles.css — visual design and responsive layout
- script.js — language switcher, menu filters, mobile navigation, reveal animation
- assets/favicon.svg — local favicon
- assets/PHOTO_SOURCES.txt — demo image source notes

Languages:
- SL, EN, IT

Important:
- Forno Lume is fictional.
- Address, phone, opening hours, menu and prices are invented.
- Photos are remote Unsplash demo images and should be replaced with real client-approved photos in a real project.
