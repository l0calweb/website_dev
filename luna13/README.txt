LUNA 13 — radical poster-style bar demo

How to open:
1. Unzip the folder.
2. Open index.html in your browser.

Files:
- index.html — page structure
- styles.css — radical poster/brutalist visual style and responsive layout
- script.js — language switcher, interactive drink board, mobile navigation, cursor glow, reveal animations
- assets/favicon.svg — local favicon
- assets/PHOTO_SOURCES.txt — demo image source notes

Languages:
- SL, EN, IT

Important:
- LUNA 13 is a fictional bar example for portfolio use.
- Address, phone, event schedule, cocktail menu and details are invented.
- Demo photos are loaded from Unsplash remote URLs in index.html.
- For a real client, replace photos, address, menu, schedule and contact data with approved real data.
