VOLT Fitness Club demo website

How to open:
1. Unzip the folder.
2. Open index.html in your browser.

Files:
- index.html — page structure
- styles.css — visual design and responsive layout
- script.js — language switcher, program info modal, schedule filters, pricing toggle, scroll progress, reveal animations
- assets/favicon.svg — local favicon
- assets/PHOTO_SOURCES.txt — demo image source notes

Languages:
- SL, EN, IT

Important:
- VOLT Fitness Club is a fictional fitness center example for portfolio use.
- Address, phone, schedule, prices and details are invented.
- Demo photos are loaded from Unsplash remote URLs in index.html.
- For a real client, replace photos, address, schedule, prices and contact data with approved real data.
